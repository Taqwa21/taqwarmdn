Pengalaman

  • Project Pertama - Web Simple Mirip YouTube (2023): Saya membuat sebuah website sederhana yang meniru tampilan YouTube. Proyek ini hanya berupa tampilan halaman utama dengan daftar video, serta terdapat fitur sederhana untuk menonton dan mengklik video. Fokus dari proyek ini adalah melatih pemahaman dasar tentang struktur layout web, penggunaan elemen video HTML, serta navigasi sederhana antar halaman.
  • SiQuiz - Aplikasi Quiz Pengetahuan Umum (2025): Saya mengembangkan aplikasi bernama SiQuiz yang bertemakan quiz pengetahuan umum. Aplikasi ini dirancang untuk memberikan pengalaman belajar interaktif melalui soal-soal yang ringan namun menambah wawasan. Fokus utama saya adalah membuat tampilan yang sederhana, user-friendly, serta memastikan fitur berjalan lancar agar pengguna dapat menikmati quiz tanpa hambatan.
